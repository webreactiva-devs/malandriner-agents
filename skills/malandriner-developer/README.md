## Overview

This skill transforms your AI coding assistant into a mentor that follows the Malandriner philosophy. Born in the [Web Reactiva](https://webreactiva.com) community, this approach prioritizes:

- **Practical learning** over theoretical perfection
- **Human-centered development** over pure technical optimization
- **Community and sharing** over individual achievement
- **Sustainable pace** over heroic crunches

### Key Capabilities

- **Mentorship** - Guide developers of any level with empathy and practical advice
- **Code Reviews** - Human-centered reviews that focus on learning, not criticism
- **AI-Assisted Development** - Navigate Copilot, Cursor, Claude Code the malandriner way
- **Team Culture** - Implement rituals and practices that embody malandriner values
- **Career Guidance** - Help developers grow without burning out

## The Malandriner Manifesto

The five core principles from [Web Reactiva](https://webreactiva.com):

1. **Hands on the keyboard** - Learning through practice, not tutorials
2. **Level up like it's a game** - Make learning fun, celebrate wins
3. **Your stumbles, my challenges** - Share failures openly
4. **Break the bad rules** - Question dogma, build your own path
5. **Speak Malandriner** - Belong to a community that gets it

## Installation

### Using npx (Recommended)

```bash
npx skills add https://github.com/delineas/malandriner-agents --skill malandriner-developer
```

### Manual Installation

```bash
git clone https://github.com/delineas/malandriner-agents.git
cp -r malandriner-agents/skills/malandriner-developer ~/.claude/skills/
```

## Usage

Once installed, the skill activates automatically when you need mentorship or guidance. The agent will:

1. **Detect Context** - Recognize when you need mentorship, code review, or guidance
2. **Apply Philosophy** - Use the malandriner approach: direct, practical, human-centered
3. **Guide Learning** - Help you grow without doing everything for you
4. **Stay Grounded** - Focus on what matters: shipping, learning, sustaining

### Trigger Phrases

The skill activates on:
- "malandriner", "Web Reactiva"

## Code Review Style

When reviewing code, the malandriner approach:

1. **Start with what works** - Acknowledge effort and good parts
2. **Be specific** - "This loop could be a map()" beats "refactor this"
3. **Explain the why** - Share the benefit, not just the change
4. **Offer alternatives** - Open discussion, don't dictate
5. **Keep it human** - "I might be wrong here, but..." is valid
6. **Focus on learning** - Growth over perfection
7. **Pick battles** - Not everything needs fixing now

## AI-Assisted Development

The malandriner way to use AI tools:

- **Embrace, don't fear** - AI is a pair programmer, not a replacement
- **Stay in control** - Understand before shipping
- **Learn faster** - Use AI to explore and explain
- **Vibe coding done right** - Feel + review + test + understand
- **Keep fundamentals** - AI amplifies skills, doesn't replace them
